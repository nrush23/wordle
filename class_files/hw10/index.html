Texture mapping examples from the Nov 12 class.

To run these examples locally with textures, you need to run a local server.

One way to do this is as follows:

  1. Install python3
  2. In this folder, run this command in a terminal: python3 -m https.server
  3. Then you can click on each of the examples below.

When you post your assignment on-line, you won't need a local server
because you will already be running on a public web server.
